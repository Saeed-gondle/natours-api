Natours API

Tour Management Platform REST API

Welcome to Natours API

This is a complete REST API for a tour management platform with advanced geospatial features. Built with Node.js, Express, and MongoDB.

API Documentation GitHub Repository

Key Features

  • Complete tour management system
  • User authentication and authorization
  • Advanced filtering, sorting, and pagination
  • Geospatial queries (tours within radius, distances calculation)
  • Reviews and ratings system with automatic calculation
  • Comprehensive API documentation

API Endpoints

Here are some of the main endpoints available:

GET/api/v1/tours - Get all tours
GET/api/v1/tours/:id - Get tour details
GET/api/v1/tours/tours-within/:distance/center/:latlng/unit/:unit - Find tours within radius
GET/api/v1/users/me - Get current user profile
POST/api/v1/users/signup - Register new user
POST/api/v1/tours/:tourId/reviews - Create tour review
