üîê Authentication

Most endpoints require Bearer token authentication. Include the token in the Authorization header:

Authorization: Bearer your-jwt-token-here
POST /api/v1/users/signup

No Authentication Required

üìù Response Format

Success Response:

"status": "success",
"results": 3,

"tours": [...]
}
}
Error Response:
{
"status": "fail",
"message": "Invalid input data"
}

Rate Limiting

API is rate limited to 100 requests per hour per IP address.

User Roles

user, guide, lead-guide, admin

Query Parameters

  • Filtering: ?difficulty=easy&price[gte]=500
  • Sorting: ?sort=price or ?sort=-createdAt
  • Field Selection: ?fields=name,price,duration
  • Pagination: ?page=2&limit=10

üöÄ Getting Started

  1. Register a new account using /api/v1/users/signup
  2. Login to get your JWT token using /api/v1/users/login
  3. Include the token in the Authorization header for protected routes
  4. Start exploring tours, creating reviews, and enjoying the API!
